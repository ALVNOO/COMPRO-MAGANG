Border Radius

Token scale saat ini, plus 3 opsi "kepribadian" visual untuk perbandingan — pilih satu arah untuk konsistensi.

Token Scale Saat Ini

None / 0px
Tabel, separator, alert-banner
--radius-sm · 4px
Tag micro, kbd, code snippet
--radius-md · 8px
Badge kecil, tooltip, chip kecil
--radius-lg · 12px
Button, dropdown item, sidebar nav-link
--radius-xl · 16px
Card, form input, tab panel
20px (praktik)
Stat card, glass card (dipakai di komponen dashboard)
--radius-2xl · 24px
Hero card, modal, floating panel besar
--radius-full · 9999px
Pill badge, avatar, chip filter, toggle

Opsi "Kepribadian" Visual

A — Sharp / Korporat
Radius kecil. Kesan formal, konservatif, enterprise.
Card → 4px
Badge → 4px
AKTIF SAAT INI
B — Balanced (Saat Ini)
Radius sedang. Profesional namun tetap approachable.
Card → 16px
Badge → pill
C — Soft / Friendly
Radius besar. Kesan modern, playful, consumer app.
Card → 24px
Badge → pill
Pilih satu kepribadian dan terapkan konsisten ke seluruh sistem. Mencampur style A+C akan terlihat tidak konsisten.
